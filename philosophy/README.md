# PocketOS 核心哲学

两条改变人生的方法论，像种子一样种在这里。

---

## [境随心转 · 身体驱动方法论](./境随心转-身体驱动方法论.md)

> 身体状态驱动心理状态，而非反过来。
> 90 秒面部泵血 + 呼吸序列 → 绕过前额叶 → 直接切换意识状态。

**核心逻辑：** 死锁检测 → 零依赖入口 → Hook 绑定 → Watchdog 监控

**一句话：** 不等状态好了再做——做了状态才会好。

---

## [无限可能 · 开放流态](./无限可能-心态论.md)

> 享受读博生活，保持开放学习的心态。
> 一切皆有可能，拥抱无限可能。

**核心信念：** 不囿于什么。流的状态。每一个可能性都是当下敞开的。

**一句话：** 读博不是轨道，是一个开放的场。

---

两条路径的交汇点：**流**。
身体在流动（血液、呼吸、热量），心态在流动（不卡在执念上）。

> 境随心转 · 斗转星移
> 无限可能 · 拥抱未来
